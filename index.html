Summary

RKParallax Effect is a iOS Library that is written in swift and provides API to create a parallax effect on UITableHeaderView.

Installation

RKParallaxEffect is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod "RKParallaxEffect"

Usage

Define a property in a UIViewController with UITableView
var parallaxEffect: RKParallaxEffect!

In viewDidAppear write
parallaxEffect = RKParallaxEffect(tableView: tableView)
parallaxEffect.isParallaxEffectEnabled = true
